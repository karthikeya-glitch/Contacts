# Contact Book Application

## Overview

Contact Book is a simple command-line Python application for storing, viewing, searching, updating, and deleting contact information. It uses a menu-driven interface to manage your contacts and makes organizing names and phone numbers easy.

## Features

- Add new contacts (name and phone number)
- View all saved contacts with their index
- Search for contacts by name (partial matches, case-insensitive)
- Delete a contact by index
- Update the details of any contact by index

## Getting Started

### Prerequisites

- Python 3.x installed on your system

### Running the Application

1. Clone this repository (or download the `contact_book.py` file).
2. Run the script in your terminal:

python contact_book.py


3. Follow the on-screen menu to manage your contacts.

## Usage

- To add a contact, choose option 1 and provide the name and phone number.
- To view all contacts, choose option 2.
- To search for a contact, choose option 3 and type in the name.
- To delete or update a contact, choose options 4 or 5 and supply the contact's index
